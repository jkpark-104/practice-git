## Instruction(git push process)

1. `mkdir practice-git`
1. `cd practice-git`
1. `git init`
1. `git remote add origin {repo address}`
1. `touch README.md`
1. git add README.md
1. git commit 
1. git branch -M main (if only your default branch name is master)
1. git push -u origin main
